# libdevice User's Guide

Prototype:


```
i32 @__nv_abs(i32 %x)
```


Description:


Determine the absolute value of the 32-bit signed integer x.




Returns:


Returns the absolute value of the 32-bit signed integer x.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
