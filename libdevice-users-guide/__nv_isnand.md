# libdevice User's Guide

Prototype:


```
i32 @__nv_isnand(double %x)
```


Description:


Determine whether the floating-point value x is a NaN.




Returns:


Returns a nonzero value if and only if x is a NaN value.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
