# libdevice User's Guide

Prototype:


```
i32 @__nv_float2int_rz(float %in)
```


Description:


Convert the single-precision floating point value x to a signed integer in round-towards-zero mode.


Returns:


Returns converted value.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
