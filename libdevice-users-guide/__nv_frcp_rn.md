# libdevice User's Guide

Prototype:


```
float @__nv_frcp_rn(float %x)
```


Description:

Returns:


Returns   1 x   .

For accuracy information see the CUDA C++ Programming Guide, Mathematical Functions Appendix, Single-Precision Floating-Point Functions section.
