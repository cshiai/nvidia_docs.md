# libdevice User's Guide

Prototype:


```
i32 @__nv_float_as_int(float %x)
```


Description:


Reinterpret the bits in the single-precision floating point value x as a signed integer.


Returns:


Returns reinterpreted value.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
