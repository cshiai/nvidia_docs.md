# libdevice User's Guide

Prototype:


```
i32 @__nv_finitef(float %x)
```


Description:


Determine whether the floating-point value x is a finite value.




Returns:


Returns a non-zero value if and only if x is a finite value.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
