# libdevice User's Guide

Prototype:


```
i32 @__nv_double2loint(double %d)
```


Description:


Reinterpret the low 32 bits in the double-precision floating point value x as a signed integer.


Returns:


Returns reinterpreted value.




Library Availability:


Compute 2.0: Yes


Compute 3.0: Yes


Compute 3.5: Yes
